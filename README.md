# 外星人入侵游戏

## 项目简介

这是我的第一个Python小项目，旨在同时学习Python的语法知识点与Git的基本使用方法。本项目是一个简单的太空射击游戏，灵感来源于经典的"外星人入侵"游戏。

## 当前版本特性

- 可控制的飞船
- 射击功能

注意：外星人敌人将在后续版本中添加。

## 操作方法

- 左右方向键：控制飞船移动
- 空格键：发射子弹

## 开发目标

1. 熟悉Python编程语言
2. 学习Git版本控制的基本使用
3. 了解游戏开发的基本概念
